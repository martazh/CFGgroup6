About Bake First Girls

Bake First Girls was created by a group of coding colleagues as an end of course project. The name is a combination of Code First Girls and our shared passion for baking. We all signed up to the Web Development course offered by Code First Girls and worked together using all our new skills to create this wonderful website using HTML, CSS and Javascript.
